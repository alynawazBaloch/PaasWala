# 🏘️ PaasWala — Your Neighborhood Community App

**PaasWala** (پاس والا — "the one nearby" in Urdu) is a neighborhood-first social platform for Pakistan. Connect with your mohalla — share updates, chat with neighbors, stay informed about local events and alerts.

## ✨ Features

- **Community Feed** — Share posts, photos, and updates with your neighborhood
- **Real-time Chat** — Message your neighbors directly
- **Events** — Create and discover local events
- **Alerts** — Emergency and community alerts
- **Polls** — Vote on neighborhood decisions
- **Marketplace** — Buy, sell, and trade within your community
- **Lost & Found** — Help reunite lost items with neighbors
- **Business Directory** — Discover local businesses and services
- **Reputation System** — Earn badges and reputation points
- **Dark Mode** — Premium glassmorphism UI with emerald theme

## 🚀 Quick Start for Developers

```bash
git clone https://github.com/alynawazBaloch/PaasWala.git
cd PaasWala
npm install
npx expo start
```

### Environment Setup
Copy `.env.example` to `.env` and fill in your Firebase, Google Maps, and Brevo API keys.

### Build APK
```bash
npx eas build --platform android --profile preview
```
